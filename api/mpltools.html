mpltools¶

Tools for Matplotlib

The primary purpose of this package is to provide tools to use pre-configured styles with matplotlib.

Subpackages¶

style
Library of styles and functions for setting the style.
color
Color choice and custom colors (e.g. parameter-based color choice).

Utility functions¶

figure
Create matplotlib figure with specified aspect ratio and scale.
figsize
Calculate figure size based on aspect ratio and scale.

Attributes¶

styles
Available matplotlib styles.
config
Dictionary of package configuration settings.
mpltools.figsize([aspect_ratio, scale, width]) Return figure size (width, height) in inches.
mpltools.figure([aspect_ratio, scale, width]) Return matplotlib figure window.

figsize¶

mpltools.figsize(aspect_ratio=1.3, scale=1, width=None)¶

Return figure size (width, height) in inches.

Calculate figure height using aspect_ratio and default figure width.

Parameters :

aspect_ratio : float

Aspect ratio, width / height, of figure.

scale : float

Scale default size of the figure.

width : float

Figure width in inches. If None, default to rc parameters.
